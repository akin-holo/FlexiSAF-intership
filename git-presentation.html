What is Git

What is Git

Git is a distributed version control system that helps developers track changes in their code, collaborate with others, and manage project history efficiently.

Configuration

Configuration

Configuration involves setting up user credentials to ensure that contributions are properly attributed in the version control system.

Configuration

Configuration involves setting up user credentials to ensure that contributions are properly attributed in the version control system.

Example:
                    
                        git config --global user.name "My Name"
                        git config --global user.email "addressname@email.com"

Setting Up the Git Control System

Setting Up the Git Control System

Setting up a Git control system begins with initializing a project repository and includes several key commands:

Setting Up the Git Control System

// initialize a Git repository

                    
                    git  init

// Stage files for commit

                    
                    git  add .

// Commit changes with a message

                    
                    git commit -m "message"

// Add a remote repository

                
                Git  remote add origin [repository-url]

// Push changes to the remote repository

                    
                    git  push - u origin

Other Git Commands

// Create a new branch

                    
                    Git  branch [name-of-branch]
